# Matriz de Causa e Efeito

## Descrição

Este arquivo corresponde a uma **Matriz de Causa e Efeito (MCE)**.

A matriz inicia efetivamente na **página 16** do documento original.

O nome do arquivo identifica automaticamente:

-   Projeto
-   Sistema
-   Disciplina

------------------------------------------------------------------------

## Identificação por Código

### Projeto

-   **3010.2J** → Projeto **P-80**
-   **3010.2P** → Projeto **P-83**

### Sistema

-   **1200** → **Topside**
-   **1351** → **Hull**

### Disciplina

-   **KES-001** → **Shutdown**
-   **KES-002** → **Fogo e Gás**

------------------------------------------------------------------------

## Estrutura de Interpretação do Nome do Arquivo

O nome do arquivo deve ser interpretado combinando os códigos acima para
identificar:

-   Projeto correspondente
-   Sistema associado
-   Disciplina da matriz

Exemplo:

`3010.2J_1200_KES-001`\
→ Projeto P-80\
→ Sistema Topside\
→ Disciplina Shutdown
